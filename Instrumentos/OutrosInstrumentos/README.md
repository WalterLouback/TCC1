# Outros intrumentos
1. Base de Código JavaScript
Conjunto de funções e classes extraídas de repositórios públicos do GitHub

Trechos de código selecionados que servirão como base para geração de documentação

Critérios de seleção dos códigos (complexidade, tamanho, domínio)

2. Prompts Padronizados
Templates de prompts utilizados para solicitar documentação JSDoc ao GitHub Copilot, ChatGPT e Amazon CodeWhisperer

Instruções específicas dadas a cada ferramenta para garantir comparação justa

Formato e estrutura dos prompts

3. Scripts de Avaliação
Scripts para calcular métricas automáticas:

BLEU (Bilingual Evaluation Understudy)

ROUGE (Recall-Oriented Understudy for Gisting Evaluation)

METEOR (Metric for Evaluation of Translation with Explicit ORdering)

CodeBLEU (específica para código)

4. Configurações de Ferramentas de Análise Estática
Arquivos de configuração do SonarQube para análise de qualidade

Configurações do ESLint adaptadas para verificar conformidade da documentação

Regras e thresholds definidos para avaliação


5. Planilhas de Coleta de Dados
Templates para registro de resultados das métricas

Formulários para comparação entre as ferramentas

Tabelas para análise estatística dos resultados

6. Protocolo de Experimento
Documento detalhando o processo experimental passo a passo

Condições controladas para geração da documentação

Metodologia de avaliação e critérios de comparação
